# Auto-Browser
Overview
Built a simple browser automation system that automates login, scrolling, and form submission on a local test application.
Tech Stack Used
Backend

Flask - Lightweight Python web framework for the test application
Python 3.x - Core programming language

Automation:
Selenium WebDriver - Browser automation framework
ChromeDriver - Chrome browser driver for Selenium

Frontend:
HTML/CSS - Simple UI for the test application
JavaScript - Basic form handling and interactions

Key Features:
Realistic delays between actions (time.sleep)
Anti-detection configuration to disable automation flags
Error handling with try-except-finally blocks
Clean browser shutdown after execution
